# GreatJob

## サイト概要
### サイトテーマ
タイムカードに日報を合わせた勤怠管理アプリ

### テーマを選んだ理由
前職の出退勤のシステムと日報が使いずらく感じていたので自分なりにもっと使いやすいシステムを作りたかったからです。

### ターゲットユーザ
バイトやお仕事をしている人々

### 主な利用シーン
出退勤時（退勤時のその日１日の作業日報を書く際）
